# Introduction

---

This Interactive Style Guide site is intended to serve as a reference for those who will be building, maintaining and/or extending the existing code.

This guide contains front-end templates (HTML, CSS, JS) along with visual design assets necessary for developing additional screens.

All data portrayed in the screen comps or examples included is for placement only. This guide does not cover detailed branding guidelines, business requirements, or user support. Any data or functionality depicted is for example purposes only. For actual data and functionality, please refer to the wireframes or the UI specifications document.
